# 🎮 Tic Tac Toe (XOXO) Game

An interactive and responsive **Tic Tac Toe (XOXO)** browser game built using **HTML, CSS, and JavaScript**.  
This project demonstrates core front-end development skills including DOM manipulation, event handling, and game logic implementation.

The goal of this project was to strengthen my JavaScript fundamentals and understand how real-time user interactions work in web applications. 🚀

---

## ✨ Features

- ✅ Two-player gameplay (X vs O)  
- 🏆 Real-time winner detection  
- 🤝 Draw match detection  
- 🔄 Game reset functionality  
- 🎨 Clean and responsive UI  
- ⚡ Smooth user interaction  

---

## 🛠️ Technologies Used

- 🌐 HTML5  
- 🎨 CSS3  
- ⚡ JavaScript (Vanilla JS)  

---

## 🧠 Concepts Practiced

- DOM Manipulation  
- Event Listeners  
- Conditional Logic  
- Functions & Arrays  
- Game State Management  

---

## 🎯 How to Play

1. Player X starts the game  
2. Players take turns marking empty cells  
3. The first player to align 3 marks (row, column, diagonal) wins  
4. If all cells are filled and no winner → It's a Draw
